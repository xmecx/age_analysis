# Аналіз вікової аудиторії

## Опис
Програма збирає вік користувачів і аналізує аудиторію.

## Примітка
Проєкт створено з навчальною метою.

## Що робить програма:
- кількість людей
- середній вік
- мінімальний та максимальний вік
- кількість повнолітніх

## Як запустити
1. Відкрийте термінал у папці проєкту
2. Виконайте команду:
   python main.py

## Приклад роботи
Введіть вік (0 — стоп):
18
25
30
0

## Висновок
Якщо середній вік >= 21 — аудиторія доросла.
Якщо середній вік < 21 — аудиторія молода.
